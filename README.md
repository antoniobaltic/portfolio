# antoniobaltic.xyz

Personal developer portfolio. One page, no dependencies, no build step.

## Stack

- HTML + CSS + vanilla JS
- [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono) via Google Fonts

## Structure

```
index.html   markup
style.css    styles + animations
main.js      typewriter sequence + scroll reveals
```

## Deploy

Drop the three files anywhere — Vercel, Netlify, GitHub Pages, or a plain static host. No config needed.
